Trabalhando com images, camadas de images e mais

Images, union file systems, fazer camadas sobre as alterações

docker image ls (lista todas as images)

docker image hisroty image:version (Historico da imagem ex: docker image nginx:latest, mostra camada de alterações feitas em uma imagem)

docker image inspect imageNameOrId (Historico da imagem ex: docker image inspect nginx, mostra dados mais detalhados da imagem)
