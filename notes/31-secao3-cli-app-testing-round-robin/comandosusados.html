Trabalhando com networking dns nos containers (como os container encontram um ao outro) e usando DNS Round Robin, balanciamento com 2 containers

DNS round Robin

docker network ls (lista todas as networks criadas)

Executados na aula

docker container run -d --network my_app_network --network-alias search elasticsearch:2 (criar dois containers com imagem do elastic search versao 2)

docker container run --rm --network my_app_network alpine nslookup search (criar um container com imagem do linux alpine e roda um nslookup no network alias "search" que foi criado)

docker container run --rm --network my_app_network centos curl -s search:9200 (criar um container com imagem do linux centos e roda um curl no network alias "search" que foi criado)
