Trabalhando com networking dns nos containers (como os container encontram um ao outro)

Comandos utilizados

docker network ls (lista todas as networks criadas)

Executados na aula

docker container run -d --name my_nginx --network my_app_network nginx (criando mais um container e adicionando na network acima)

docker container exec -it my_nginx ping new_nginx (Verificando se o container my_nginx consegue fazer um ping na new_nginx)
