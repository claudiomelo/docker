Trabalhando com images, criando imagens e dockerfile

Criar imagens com dockerfile:
https://docs.docker.com/engine/reference/builder/

Executados na aula

docker build -f some-dockerfile (executa um dockerfile, -f é para especificar um arquivo que não tenha o nome default de "Dockerfile")

Primeiro comando do dockerfile é o FROM, "FROM repo:image"

docker image build -t imageNameIWant . (criar uma imagem a partir de um docker file, o . significa criar no diretorio atual, onde deve conter o dockerfile)

OBS: sempre deixar os comandos que precisam ser sempre executados ou que podem mudar com frequencia no fim do dockerfile, boas praticas
