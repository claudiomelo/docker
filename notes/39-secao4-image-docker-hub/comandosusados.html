Trabalhando com images, docker hub, tags

Criar tags e subir imagens no dockerhub

Executados na aula

docker image tag --help (help do comando image tag)

docker image ls (lista as tags de imagens)

docker pull userName/repository (Fazer pull de uma imagem, nas imagens oficiais não tem o username)

docker pull userName/repository:tag (Fazer pull de uma imagem pela tag, nas imagens oficiais não tem o username)

docker image tag sourceImage[:tagOpcional] targetImage[:tagOpcional] (Adicionar uma tag em uma imagem, se não especificar a tag ele coloca a tag latest)

docker login (Fazer login no docker hub para poder fazer o push)

docker logout (Fazer logout no dockerhub)

docker image push imageRepository (Fazer push de uma imagem para o dockerhub)

cat .docker/config.json (Arquivo com configs do docker)
